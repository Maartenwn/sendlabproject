How to use?

Add componenents

To add an component press the button with the component you want to add.

Remove componenents

To remove an component press the button with the component you want to remove.

Drag componenents

To drag components press: 'Ctrl + Left click' and drag the component to another component to switch.

Note

There can only be 5 components selected at a time.

Buttons explained

GPS

The button GPS shows a map with the GPS coordinates of the solarboat.

Livestream

The Livestream button shows a youtube livestream from the solarboat team.
